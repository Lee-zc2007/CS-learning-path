#函数封装与绑定
'''
std::function
std::bind
std::mem_fn
'''  
将可调用对象（函数指针，仿函数，lambda）进行封装方柏霓使用  
function类模板定义  
'''
template <class R, class... Args>//Args是可变模板参数
class function<R(Args...)>;
std::function<R(Argss...)> fnname = target;//实际使用
'''
使用的时候用函数签名填充<>  
封装类的成员函数时候模板参数小括号里面第一个参数是类的引用，对应的成员函数或者成员变量也要显式引用（不在全局，在类的定义下，不能直接取地址）
`std::function<float(Linear&,float)> mf = &Linear::f;`
类型擦除模式
'''
std::map<char, std::function<double(double,double)>> calculator{
    {'+', add},//函数指针
    {'-', Subtract}//仿函数struct
    {'*', [](double a, double b)->double{return a*b;}}//lambda表达式
};

std::cout<<calcutlator['+'](12.0,13.0)<<std::endl;
//擦除封装对象的具体类型，保留一样的接口
'''
封装类的成员用std::mem_fn
参数是指向类成员的指针，返回值是可调用的包装器
'''
template<class M, class T>
mem_f(M T::*pm) noexpect;

auto memfn = std::mem_fn(&Foo::calculate);//举例
'''

std::bind是函数模板，用来生成函数调用的转发包装器（普通转发会丢失参数类型）
'''
template<class F, class... Args>
bind(F&& f, Args&&... args);

auto f = std::bind(sum,1,2,3);//使用bind将sum和参数绑定
int res = f();
'''

也可以用std::placeholders中的占位符来代替被绑定的实际参数，可以在调用的时候再传入实际参数
'''
#include <functional>
using namespace std::placeholders;

auto f = std::bind(sum,1,_1,3);
int res = f(5);//res = 9
'''

将变量作为参数绑定默认按值，绑定后再修改变量并不会改变绑定对象的参数，可以用std::cref函数，返回一个变量的引用封装器
'''
auto f = std::bind(sum,1,2,std::cref(n));
